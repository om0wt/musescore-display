    transposeToKey

    • transposeToKey(targetFifths: number, direction?: "up" | "down" | "auto"): void

      Transpose the score to a target key (specified as circle-of-fifths position) and re-render. The transposition is calculated relative to the original key.

      Parameters

      • targetFifths: number

        Circle-of-fifths position of the target key (-7 to +7). For example: 0 = C major/A minor, 1 = G major/E minor, -1 = F major/D minor.

      • direction: "up" | "down" | "auto" = "auto"

        "auto" picks the shortest interval (default), "up" always transposes up (0..+11), "down" always transposes down (-11..0).

      Returns void

      • Defined in MuseScoreDisplay.ts:178
